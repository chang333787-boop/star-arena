## 1단계. 공통 골격 — 캠페인·코옵이 같이 쓰는 구조물 시스템

### 1.1 설계 원칙 (PRD 8.2 준수)
- **고정형**: 구조물 위치·적 등장 순서·경로는 스테이지에 하드코딩. 무작위성은 강화 3택1에만 둔다.
- **공통 뼈대**: 혼자 도전 캠페인 = 협동 PVE의 host 로직 그대로(v1.20 방식) → 구조물도 같은 코드 하나로 두 모드 커버. 차이는 배율·목숨 수치만.
- **읽히는 충돌**: 구조물은 전부 사각 충돌(OBSTACLES와 동일 규칙). 파괴되면 충돌 제거 + 잔해 0.4초.

### 1.2 목표(objective) 5종 — 스테이지 데이터에 `objective` 필드 추가
| 목표 | 동작 | 실패 조건 | 1차 구현 |
|---|---|---|---|
| `waves` (현행) | 웨이브 전멸 | 전원 사망+부활 불가 | ✅ 이미 있음 |
| `defend` 방어 | 방어 수정 75초 보호(타이머) | 수정 HP 0 또는 전멸 | **✅ 이번 대상** |
| `blockade` 봉쇄 | 몬스터 문 N개 파괴(문이 계속 적 생성) | 전멸(시간제한 없음) | **✅ 이번 대상** |
| `escort` 호위 | 씨앗이 경로 이동, 근처에 플레이어 있어야 전진 | 씨앗 HP 0 | 2차(경로 이동 로직 필요) |
| `hold` 거점 | 거점 원 안 유지 45초 | 전멸 | 2차(점령 게이지 UI 필요) |

### 1.3 구조물 엔티티 모델 (신규 코드 골격)
```
tStructures = { id: { type, x, y, w, h, hp, maxHp, alive } }   // host 권위
type: "crystal_core"(방어 수정) | "monster_gate"(봉쇄 문) | "cushion_wall"(파괴 벽)
```
- **충돌**: 스테이지 시작 시 `OBSTACLES`에 push, 파괴 시 제거(기존 moveEntity/hasLineOfSight가 자동 반영 — 신규 판정 코드 불필요).
- **피격**: 적 탄환·접촉 → `crystal_core`만 피해. 플레이어 탄환 → `monster_gate`(봉쇄 모드)·`cushion_wall`만 피해. 아군 오사 규칙 그대로.
- **어그로**: 방어 스테이지에서 잡몹 1/3은 스폰 시 `focusStructure=true`로 수정 직행(나머지는 플레이어 추적 — 현행 AI 그대로). 보스는 구조물 무시. *(v1.23: 50%→1/3, 코덱스 회차8 — 입문 스테이지 실패 압력 완화)*
- **회복 금지**: 모아 패시브/기물/궁, 회복 아이템은 파이터 대상이라 구조물에 자연 미적용(코드 확인 완료 — env.allies는 tFighters만 순회).
- **네트워크**: `st.structs=[{t,x,y,hp,mh}]` (숫자만, 15fps 패킷에 추가 — 구조물 ≤ 6개라 페이로드 영향 미미).
- **HP 스케일(PRD 19.1)**: 수정 500+120×(N-1) · 문 180+50×(N-1) *(v1.23: 6인 문 포커싱 대비 배율 추가, cap 4 유지)* · 쿠션 벽 120(고정).

## 4단계. 패시브 × 구조물 스테이지 궁합 판정

| 패시브 | 방어/봉쇄에서의 역할 | 판정 |
|---|---|---|
| 럭키 무한 탄창 | 장전 끊김 없이 문/웨이브 지속 화력. 초심자 안전판 | **유지** — DPS 자체는 6인 중위권, 시스템 우회이지 수치 우위 아님 |
| 달이 원거리 +30% | 수정 뒤 고정 포대 = 저격수 정체성의 완성 무대 | **유지** — 방어전이 달이 하이라이트가 되는 게 설계 의도 |
| 시고니 등 뒤 +20% | 수정으로 달려드는 몹의 등이 자연 노출 → 방어전에서 상시 발동 우려 | **조건부 관찰** — v1.22 시뮬에서 방어전 DPS 1위가 20% 이상 벌어지면 PVE 한정 15%로 감쇠 |
| 눈꽃 Z 20% 둔화 | 문 앞 병목·수정 앞 저지선 — 봉쇄/방어 최고 유틸 | **유지** — 둔화 중첩 없음 규칙이 이미 상한. 보스 면역 유지 |
| 모아 오오라/주변 힐 | **수정 회복 불가(PRD)** 라 구조물전 파워 급증 없음. 팀 유지력 담당 | **유지** — 구조물 회복 금지 규칙을 코드로 보장(1.3) |
| 별골렘 막으면 궁 충전 | 문/수정 앞 몸빵 → 게이지 수급 최고. 가동률 60% | **조건부 관찰** — 방어전에서 블랙홀 순환이 8초 미만이면 gaugePerBlock 0.6→0.45 |
- 종합: 구조물 스테이지가 **서포터·탱커·저격수의 무대**가 되는 방향이라 리워크 취지와 일치. 과유리 후보는 시고니·별골렘 2건만 시뮬 지표로 판단.
