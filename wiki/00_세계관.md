## 1. 한 줄 이야기 (로그라인)

> 별빛으로 농사짓는 **별마중 마을**의 별빛이 점점 흐려진다.
> 할머니의 밭을 물려받아 이사 온 외계 고양이 **시고니**가,
> 별빛을 훔쳐가는 **먹구름단**을 호미 한 자루부터 막아내고,
> 챕터 끝에서 상징 도구 **반짝 부메랑**을 완성한다.

이 이야기는 별빛 아레나에서 부메랑을 던지던 그 시고니의 **어린 시절 기원담**입니다.

## 2. 주인공 — 시고니

- 외계에서 온 고양이. 할머니가 남긴 밭과 농가를 물려받아 별마중 마을에 이사 왔다.
- 시작은 **맨손 레벨 1**. 촌장님께 **호미**를 선물 받고, 창고에서 **낡은 부메랑**을 찾아내고, 마지막에 대장간에서 **반짝 부메랑**을 만든다.
- 낮에는 밭을 갈고, 밤에는 마을을 지킨다 — 농사와 모험이 자꾸 서로를 부른다.

## 3. 별마중 마을 (외부 36×16칸)

별이 마중 나오는 마을이라서 '별마중'. 마을 사람들은 별빛을 받아 농사를 짓고 도구를 만든다.

| 장소 | 설명 |
|---|---|
| 🏠 할머니의 농가 | 시고니의 새집. 침대(자면 저장+하루 진행)·창고 상자 · **🔒 잠긴 지하실 문(여백 ②)** |
| 🌾 시고니 밭 (3×4) | 할머니가 남긴 밭. 나중에 확장 가능 |
| 🛒 잡화점 (모모) | 씨앗·소모품 사고팔기 — 처음부터 열려 있음 |
| ⚒️ 대장간 (볼트) | 도구 제작·강화 — 마을 부탁(mq04)을 도우면 열림 |
| 🏛️ 촌장집 (온별) | 메인 퀘스트의 시작점 |
| ⛲ 광장·우물 | 게시판(의뢰·미해결 사건)과 별우물(쓰러지면 여기서 다시) · **우물 밑 노랫소리(여백 ⑥)** |
| ✨ 별광맥 | 마을 별빛의 원천 — 먹구름단이 노린다(mq06 방어전) |
| 🧱 빈 부지 2곳 | **학생 건물 슬롯** — 우리 반이 지을 건물 자리 |
| ➡️ 동쪽 출구 | 이슬별 들판(티어1) → 가랑비 숲길(티어2) → 먹구름 요새(티어3, 방 3개) |

## 4. 세계 규칙 3가지 ⭐ (모든 시트가 지켜야 해요)

1. **아무도 죽지 않아요 — 별가루 귀환.** 쓰러진 몬스터도, 시고니도, 별가루가 되어 하늘로 반짝이며 돌아가요. 시고니는 별우물에서 다시 일어나요(벌점 없음). 먹구름단 단원은 "뿅!" 연기와 함께 도망가요.
2. **별빛은 마을의 생명이에요.** 농사도, 도구도, 등불도 별빛으로 움직여요. 별빛이 흐려지면 마을 전체가 시들어요 — 그래서 이 사건이 큰일이에요.
3. **먹구름단은 물건만 훔쳐요.** 사람은 절대 다치게 하지 않아요. 훔친 별빛은 요새의 '알'에게 가져가요 — 왜인지는 아무도 몰라요(최대 여백 ①).

## 5. 먹구름단 ☁️

### 5.1 계급도

```
      ❓ 그분(?)          ← 정체 미공개 (챕터 1 최대 비밀)
         │
      👑 뭉게대왕          ← 챕터 1 보스 (먹구름 요새)
         │
   🌦️ 날씨 간부단         ← 간부 자리 여러 개 = 학생 확장 슬롯!
   (기본 1명: 안개 간부 보슬보슬)
         │
      🪣 부슬이들          ← 단원. 양동이 투구, 2인 1조로 대사를 반쪽씩 나눠 말함
```

### 5.2 기본 간부 예시 — 안개 간부 보슬보슬

| 칸 | 내용 |
|---|---|
| 이름(name) | 보슬보슬 |
| 날씨(weather) | 안개 |
| 입버릇(catchphrase — 유머 필수) | "앞이 안 보이지? …나도 안 보여…" |
| 부하 구성(minion_palette) | 기존 몬스터 2~3종 색놀이 |
| 소굴(lair) | 이슬별 들판 야영지(mq05) |
| 기술(pattern) | 보스 기술 재조합 |
| 훔치는 것(steals) | 별씨앗 |
| 웃긴 약점(weakness = 전투 힌트) | (전투 중에 힌트로 등장) |
| 마지막 말(last_words) | "그분이 기다리셔…" |

### 5.3 간부 확장 틀 — 새 간부를 만들려면 이 9칸을 채워요

`이름 · 날씨 · 입버릇(꼭 웃기게) · 부하 구성(기존 몬스터 2~3) · 소굴 · 기술(보스 기술 재조합) · 훔치는 것 · 웃긴 약점(=전투 힌트) · 마지막 말`

> 예: 천둥 간부? 무지개 간부? 이슬비 간부? — **날씨 하나 = 간부 하나**, 자리는 남아 있어요.

## 7. 열린 여백 7가지 ❓ — '미해결 사건' 게시판과 1:1

여백의 규칙(§3.4): **여백은 항상 흔적과 함께** 남는다 · 우리 반 답이 채택되면 **데이터 행이 교체되어 세계가 진짜로 바뀐다** · 여백이 채워지면 **새 여백을 보충**한다.

| # | 미해결 사건 | 남아 있는 흔적 | 학생이 이어 쓸 질문 |
|---|---|---|---|
| ① | 요새의 알과 '그분' | 먹구름 요새 깊은 곳의 커다란 알. 훔친 별빛이 전부 여기로 | 알 속에는 무엇이 있을까? '그분'은 누구일까? 먹구름단은 왜 "우리도 빛이 필요했을 뿐"이라고 했을까? |
| ② | 잠긴 지하실 | 할머니 농가의 잠긴 문. 열쇠는 어디에도 없다 | 할머니는 지하실에 무엇을 숨겼을까? 열쇠는 누가 갖고 있을까? |
| ③ | 찢어진 유품 지도 | 할머니 유품 속, 반쪽만 남은 지도 | 나머지 반쪽은 어디에? 지도가 가리키는 곳엔 무엇이? |
| ④ | 빛나지 않는 별조각 | mq05에서 얻는, 유일하게 빛나지 않는 별조각 | 왜 이 조각만 빛나지 않을까? 어디서 떨어진 조각일까? |
| ⑤ | 따라오는 젤리 1마리 | 싸운 뒤에도 도망가지 않고 시고니를 졸졸 따라오는 그림자 젤리 한 마리 | 이 젤리는 왜 따라올까? 이름을 지어 준다면? 친구가 될 수 있을까? |
| ⑥ | 우물 밑 노랫소리 | 밤에 광장 우물에 귀를 대면 들리는 희미한 노래 | 누가 부르는 노래일까? 무슨 노래일까? 우물 밑에는 무엇이 있을까? |
| ⑦ | 할머니 일기의 찢어진 마지막 장 | 농가에서 발견한 일기 — 마지막 장만 찢겨 있다 | 할머니는 어디로, 왜 떠났을까? 마지막 장에는 무엇이 적혀 있었을까? |

## 8. 메인 이야기 7비트 요약 (챕터 1)

| # | 제목 | 무슨 일이 일어나나 | 농사와의 연결 | 남는 '왜?' (수업 발문) |
|---|---|---|---|---|
| mq01 | 별마중 마을 도착 | 농장 상속. 촌장님께 **호미**를 받는다 | 잡초 정리·첫 파종 | 할머니는 어디로, 왜 떠났나? |
| mq02 | 밤손님 | 그림자 젤리가 밭을 습격 — 호미로 첫 전투 | 물주기→취침→**첫 수확이 완료 조건** | 온순하던 젤리들이 왜 내려오나? |
| mq03 | 별조각 도둑 | 부슬이들이 별조각·별씨앗을 훔쳐 간다. 창고에서 **낡은 부메랑** 발견 | 별씨앗 도난(작물 잠금) | 훔친 별빛을 어디에 쓰나? |
| mq04 | 마을의 부탁 | 의뢰 3건 해결 → **대장간 개방**·스킬 [빙글 베기] | 의뢰 1건은 작물 납품(교차점 1) | 먼별은 먹구름단을 어떻게 잘 아나? |
| mq05 | 안개 간부 보슬보슬 | 야영지 간부전. 별씨앗 회수 → '별방울 콩' 해금·전직 개방 | **회수한 씨앗을 심고 수확해 바쳐야** 다음 문이 열림 | "그분이 기다리셔…" |
| mq06 | 별광맥 방어전 | 밤의 습격 방어전 — **낮에 심은 빛나무 묘목 수만큼 방어 조명 생성**(교차점 3). 별조각×3 획득 | 낮의 밭이 밤의 전장 | 왜 훔치기만 하나? |
| mq07 | 먹구름 요새 — 뭉게대왕 | 입장권은 **원정 도시락(별방울 콩×3)**(교차점 2). 직전에 **반짝 부메랑 제작**. 알 발견 → 보스 격파 → 마을 축제 | 보스로 가는 문이 밭에서 시작 | "우리도… 빛이 필요했을 뿐이야." 알 속엔? |
