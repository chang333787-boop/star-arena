## 0. 설계 원칙 — "시트 규칙을 지켰는데 반려" 0건

학생 시트는 숫자 대신 **⭐색칠**만 쓴다(§8.2). 변환표가 ⭐를 절대치로 바꾸고, 파워버짓이 최종 상한을 지킨다. 핵심은 **역산 정합**:

> **⭐예산을 지킨 모든 합법 조합은 파워버짓을 자동 통과하도록 변환표를 역산해 둔다.**
> 검증기의 클램프·스케일다운(§7.4 ②)은 시트 외 경로(직접 JSON 등)용 안전망이지, 합법 시트가 걸리는 일은 구조적으로 없다.

이를 말이 아니라 **전수 검산**으로 보증한다 — 아래 각 절의 결과표가 그 증거이며, 언제든 재실행 가능:

```bash
python3 wiki/tools/verify_conversion.py   # exit 0 = ALL PASS
```

### 파워버짓 공식 4종 (PRD §7.5 — 재수록)

| 대상 | 공식 | 상한 |
|---|---|---|
| 몬스터 | `hp/10 + atk×2 + speed/30` | `20 + 티어×15` (마을0=20 / 들판1=35 / 숲길2=50 / 요새3=65) · 가변 이속은 시간가중 평균 · **isBoss=면제(교사 직접 검수)** |
| 도구 | `편도 DPS = dmg/cd` | `15 + levelReq×3` (부메랑류는 편도 기준 — 왕복은 숙련 보상) |
| 스킬 | `dmgMul×100 / cd` | `60` (게이지·패시브형은 공식 비적용 — 교사 검수 전용) |
| 작물 | (변환표 아님 — 규칙 검증) | `sellPrice ≤ seedPrice×3` **그리고** 순이익/일 8~14 밴드(§5.3) |

## 1. 몬스터 변환표 (PRD 부록 **확정** — 변경 금지)

**시트 구성(§4.3)**: 이름 / 그림 / 체력⭐ / 공격⭐ / 빠르기⭐ (**⭐예산 7**) / 행동카드 1~2장 / 떨어뜨리는 것 / 사는 곳.

|---|---|---|---|
| ⭐1 | 12 | 2 | 40 |
| ⭐2 | 25 | 4 | 70 |
| ⭐3 | 45 | 6 | 110 |

- 설계 근거: 곡선이 위로 갈수록 가파른(체감 성장) 수열이면서, 예산 7의 **이론 최악 조합 (1,5,1)의 버짓 = 28.5**가 티어1 상한 35 아래 **여유 6.5**로 들어오게 역산. atk 최대 13은 상한표(1~25)와 난이도 헌장(한 방 ≤ HP 10% — 권장 레벨 기준) 안.
- 학생 몬스터의 기본 서식은 들판(티어1)으로 접수 — 티어1을 통과하면 상위 티어(35<50<65)는 자동 통과이므로 서식 상향 배치도 안전.
- 행동카드 메뉴(9종)는 시트 양식(RPG-0 후속)에서 확정. 현재 확보된 프리미티브: chase / hop(신규) / turret / dash / ghost / fan(비보스 배선 확장) / blink(신규) / frontGuard — 8종 + 여유 1(§4.3·§10.1).
- 드랍(떨어뜨리는 것)은 상한표 별도 적용: `exp 1~30 · gold max ≤ exp · chance ≤ 0.5`(보상 민맥싱 가드 — §7.1). ⭐버짓과 독립.

## 3. 스킬 변환표 (신규 도출 — 같은 역산 원칙)

**시트 구성(PRD §4.5 확정)**: 형태 3종 템플릿(⭕원형 / ➡️전방 / 🎯던지기) 중 1 동그라미 + **세기⭐ / 기다림⭐ 예산 6**.

| ⭐ | 세기 → 배수 dmgMul | 기다림 → 재사용 cd(초) |
|---|---|---|
| ⭐1 | ×0.8 | 10.0 |
| ⭐2 | ×1.0 | 8.0 |
| ⭐3 | ×1.2 | 6.0 |
| ⭐4 | ×1.5 | 4.5 |
| ⭐5 | ×1.9 | 3.0 |

- **기다림⭐는 많을수록 기다림이 짧다**("참을성이 덜 필요해요") — 시트에 이 문장을 명기해 혼동 차단. ⭐1=10초 … ⭐5=3초.
- 형태는 버짓 무관(고정 템플릿 파라미터): ⭕원형 = around·반경 110(빙글 베기 형) / ➡️전방 = front(별똥별 형·넉백 없음) / 🎯던지기 = throw·**탄 1갈래**(다갈래는 공식이 과소평가하므로 기본 스킬 전용 — 공식 정직성).
- 역산 근거: 전수 최악 26.7이 상한 60의 절반 이하이며, **기본 스킬 밴드(세갈래 10 · 빙글 24 · 별똥별 25)와 같은 구간**에 학생 스킬(8.0~26.7)이 정확히 겹친다 — 학생 스킬이 기본 스킬을 무력화하지 않고 어깨를 나란히 한다.
- 게이지형(반짝 질주형)·패시브형은 공식 비적용 — **교사 검수 전용**(§7.5). 학생 시트는 재사용(cd)형만. acquire(배우는 시점)는 승인 패널에서 교사가 지정.

## 5. 작물 — 참고(변환표 아님)

작물 시트(차시 4)는 ⭐변환이 아니라 **규칙 검증**: `sellPrice ≤ seedPrice×3` + **순이익/일 8~14 밴드**(성장이 길수록 이익/일↑ — 기본 5종이 8.0 < 9.0~9.5 < 13.8~14.0으로 스스로 규칙 준수, §5.3). 판매 불가 특수 작물(빛나무 묘목형)은 교사 검수 전용. 시트 양식 설계 시 이 문서에 변환 규칙을 추가한다.

## 6. 유지보수 규칙

- 이 문서의 수치를 바꾸려면: ① PRD 개정 → ② `wiki/tools/verify_conversion.py` 상수 수정 → ③ 재실행 ALL PASS 확인 → ④ 이 문서 결과표 갱신 → ⑤ `harness_rpg.js` 이식분 동기(RPG-1 이후). 순서 역행 금지 — **검산 없는 수치 변경은 반영 금지**.
