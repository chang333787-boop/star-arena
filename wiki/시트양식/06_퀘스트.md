## 📄 뒷면 인쇄 — 규칙·전사 도우미

### 규칙 (§6 · §7.1)
- goal 5타입: `kill / collect / talk / deliver{item,to} / harvest` — **배달(deliver)만 '받는 사람' 2번째 칸이 필요**
- 목표물·주는 사람은 **실제 존재하는 카드**여야 해요 (게시판에서 확인 — 유령 id 금지)
- 대사 줄당 40자 · 퀘스트는 단일 goal · 이어지는 이야기는 `requires` 체인으로
- 보상 참고 눈금(기본 사이드 퀘스트): 30~60exp · 40~80G 수준 — 별⭐을 이 눈금 안에서 교사 환산

### 전사 도우미 (선생님용 — 시트 칸 → JSON 필드 1:1)

| 시트 칸 | JSON 필드 (`RPG_DB.quests`) | 비고 |
|:---|:---|:---|
| ID | `id` | `Q-03` → `stu_quest_3` (제출 시점 영구 확정) |
| 퀘스트 이름 | `name` | |
| 누가 부탁 | `giver` | N카드 id — npc.questIds는 부팅 시 자동 파생 |
| 종류 동그라미 | `goal.type` | 물리쳐요=kill · 모아요=collect · 말걸어요=talk · 배달=deliver · 수확=harvest |
| 무엇을/누구를 | `goal.target` 또는 `goal.item` | kill/talk=target · collect/deliver/harvest=item |
| 몇 개 동그라미 | `goal.count` | 1~5 |
| 받는 주민 | `goal.to` | deliver 전용 2인자 |
| 먼저 깰 퀘스트 | `requires` | 선택 |
| 시작/끝 대사 | `dialogStart` / `dialogEnd` | 빈칸 채운 완성문 |
| 다음 힌트 | `nextHint` | 선택 항목 (25분 상한 — 없어도 통과) |
| 보상⭐ | `reward{exp,gold,itemId}` | gold ≤ exp 원칙 준용 · 물건 보상은 실존 id만 |
