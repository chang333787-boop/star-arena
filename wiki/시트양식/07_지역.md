## 📄 뒷면 인쇄 — 규칙·전사 도우미

### 규칙 (§7.1 · §7.4)
- **학생 지도는 24×12칸 이하** (이 양식의 격자가 상한 그 자체예요)
- 검증기 검사: 막힌 방 없음(flood-fill) · **문 목적지가 걸을 수 있는 칸** · **반대쪽 문(역방향 포탈) 존재** · 전 지역 연결 그래프 도달성
- 주민 위치는 **지도가 단일 출처** — N카드에는 좌표를 쓰지 않아요
- 몬스터는 사는 곳(티어)에 어울리게 — 바닥 테마와 티어를 맞춰요

### 전사 도우미 (선생님용 — 시트 칸 → JSON 필드 1:1)

| 시트 칸 | JSON 필드 (`RPG_DB.maps`) | 비고 |
|:---|:---|:---|
| ID | `id` | `A-02` → `stu_map_2` (제출 시점 영구 확정) |
| 지역 이름 | `name` | |
| 격자 | `rows[]` (ASCII) | 종이 모눈과 1:1 — 미러 없음·풀 저장 · ■=벽, 빈칸=길, ~=물, ♣=풀 |
| 바닥 테마 | `theme` | FLOOR_THEMES 재사용 |
| 문 표 | `portals[]` | `to`는 실존 맵 id · 역방향 포탈 필수 |
| 배치표 N | `npcs[{npcId,tx,ty}]` | 격자 위 N 기호의 칸 좌표를 교사가 읽어 기입 |
| 배치표 M | `monsters[]` | 〃 |
