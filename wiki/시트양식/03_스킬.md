## 📄 뒷면 인쇄 — 상한표·전사 도우미

### 상한표·버짓 (§7.5)
- 파워버짓(스킬): `세기(dmgMul×100) ÷ 기다림(cd) ≤ 60` — 예: 빙글 베기 1.2배·5초 → 120÷5=24 ≤ 60 통과
- ⭐→숫자 변환은 `변환표_밸런스.md`(RPG-0 산출물 · 몬스터 표와 같은 역산 원칙) — **⭐예산 6을 지킨 시트는 자동 통과하도록 역산**
- 게이지형(꽉 차면 쓰는 스킬)·패시브(늘 켜져 있는 힘)는 공식 비적용 → **선생님과 함께 만드는 특별 스킬**(교사 검수 전용)
- 이름 한글 2~8자 · 설명 줄당 40자

### 전사 도우미 (선생님용 — 시트 칸 → JSON 필드 1:1)

| 시트 칸 | JSON 필드 (`RPG_DB.skills`) | 비고 |
|:---|:---|:---|
| ID | `id` | `S-04` → `stu_skill_4` (제출 시점 영구 확정) |
| 스킬 이름 | `name` | |
| 형태 동그라미 | `form` | 원형=`around` · 전방=`front` · 던지기=`throw` |
| 세기⭐ | `dmgMul` | 변환표 |
| 아낀 기다림⭐ | `cd` | 별 많음 = cd 짧음 (버짓식으로 역산) |
| 범위 동그라미 | `radius` | 좁게/보통/넓게 → 교사 환산 |
| 배우는 방법 | `acquire{by,value}` | 레벨=`{by:"level",value:N}` · 직업=`{by:"job",value:J-id}` · 퀘스트=`{by:"quest",value:Q-id}` |
| (시트에 없음) | `type`·`slot` | 학생 스킬 기본값: 액티브 · X 교체 장착 (§4.5) |
| (특별 스킬만) | `gauge` | 게이지형 — 교사 검수 전용 |
