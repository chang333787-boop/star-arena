## 📄 뒷면 인쇄 — 상한표·전사 도우미

### 상한표·규칙 (§5.3 · §7.5)
- `sellPrice ≤ seedPrice × 3` (판매값은 씨앗값의 3배까지)
- **순이익/일 = (판매값 − 씨앗값) ÷ 성장일수 → 8~14 밴드** 안에서 자동 결정 (오래 걸릴수록 밴드 위쪽: 1일≈8 → 2일≈9~9.5 → 3~4일≈13.8~14)
- 성장 판정: `wateredDays ≥ growDays` (물 준 날만 카운트 — 사멸 없음)
- 이름 한글 2~8자

### 전사 도우미 (선생님용 — 시트 칸 → JSON 필드 1:1)

| 시트 칸 | JSON 필드 (`RPG_DB.crops`) | 비고 |
|:---|:---|:---|
| ID | `id` | `C-03` → `stu_crop_3` (제출 시점 영구 확정) |
| 작물 이름 | `name` | |
| 며칠 동그라미 | `growDays` | 1~4 |
| 다시 열리나요 | `regrowDays` | 없음=`null` · 있음=`2` |
