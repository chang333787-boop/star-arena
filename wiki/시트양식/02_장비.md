## 📄 뒷면 인쇄 — 상한표·전사 도우미

### 상한표 (§7.1)
- 도구: `dmg` 1~30 · `cd` 0.35~2.0초 · 이름 한글 2~8자
- 파워버짓(도구): `편도 DPS(dmg/cd) ≤ 15 + levelReq×3` — 부메랑류는 편도 기준(왕복은 숙련 보상)
- ⭐→숫자 변환은 몬스터 표와 **같은 역산 원칙**의 `변환표_밸런스.md`(RPG-0 산출물)를 따라요 — ⭐예산 6을 지킨 시트는 자동 통과하도록 역산돼요
- 넘치면 반려가 아니라 **비율 조정**돼요: "공격력 9999 → 13으로 조정했어요" 안내가 나가요 (§7.4)

### 전사 도우미 (선생님용 — 시트 칸 → JSON 필드 1:1)

| 시트 칸 | JSON 필드 | 비고 |
|:---|:---|:---|
| ID | `id` | `E-05` → `stu_weapon_5` 또는 `stu_armor_5` (제출 시점 영구 확정) |
| 종류: 도구 | `RPG_DB.weapons` 행 | 가까이=근접(호 판정)/멀리=원거리 → `kind` |
| 종류: 옷·장신구 | `RPG_DB.armors` 행 | 옷 → `slot:"body"` · 장신구 → `slot:"charm"` |
| 세기⭐ | `dmg` | 변환표 |
| 빠르기⭐ | `cd` | 별 많음 = cd 짧음 (0.35~2.0 안) |
| 사거리⭐ | `range` (+`bulletSpeed`·`arcDeg`는 교사 기본값) | |
| 튼튼/지킴/발걸음/회복⭐ | `hpAdd` / `defAdd` / `spdMul` / `regenAdd` | |
| 값 동그라미 | `price` | 싸요~아주 비싸요 → 수지표(§5.5) 밴드 안에서 교사 환산 |
| 얻는 방법 | `source` | 드랍이면 해당 몬스터 `drops`에도 반영 |
| 레벨 동그라미 | `levelReq` | 버짓 상한이 함께 올라가요 |
| 재료 칸 | 제작 레시피(`items` 참조) | 유령 id 금지 — 실제 M카드 드랍만 |
