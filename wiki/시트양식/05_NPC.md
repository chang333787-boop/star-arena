## 📄 뒷면 인쇄 — 규칙·전사 도우미

### 이름·대사 규칙 (§3.3 · §8.4)
- 한글 2~8자 · 대사 줄당 40자
- 쓸 수 없는 이름: **기존 주인공 6인** · 마을 기본 주민(온별·모모·볼트·리프·먼별·피코) · 실존 인물·우리 반 친구 실명 (검증기 금칙 목록)
- 아무도 죽지 않아요 — 무서운 설정 대신 "별가루" 세계관에 맞게!

### 전사 도우미 (선생님용 — 시트 칸 → JSON 필드 1:1)

| 시트 칸 | JSON 필드 (`RPG_DB.npcs`) | 비고 |
|:---|:---|:---|
| ID | `id` | `N-04` → `stu_npc_4` (제출 시점 영구 확정) |
| 주민 이름 | `name` | 금칙 목록 통과 필수 |
| 역할 동그라미 | `role` | 가게 주인/부탁하는 사람/이야기꾼/꼬마 친구 |
| 평소 하는 말 ①② | `dialog[]` | 상태별 배열 — 기본 2줄부터 |
| 말버릇 | `speechTic` | |
| 파는 것 | `shop[{itemRef}]` | 실제 존재하는 카드 id만 (유령 id 금지) |
| 비밀 칸 | `secret` | 여백(§3.4) 재료 — 도감에는 비공개 |
| (시트에 없음) | 좌표 없음! | **배치(tx/ty)는 map이 단일 출처** — 지역 시트의 `npcs[{npcId,tx,ty}]`로만 |
| (자동 생성) | `questIds` | 퀘스트가 단일 출처 — 부팅 시 파생 |
