# 별빛 아레나 (Star Arena)

크롬북 Chrome에서 바로 실행되는 1인용 대 컴퓨터 아레나 슈팅 게임입니다.
외부 라이브러리·이미지·사운드 없이 순수 HTML/Canvas 한 파일로 만들었습니다.

## 플레이 방법

- **WASD** : 이동
- **방향키** : 공격 (이동과 동시·대각선·연속 발사 가능)
- **Space** : 필살기 (게이지 100%일 때)
- **P** : 일시 정지 / **Enter** : 시작·버튼 실행 / **R** : 다시 시작

90초 동안 컴퓨터(크롬)를 더 많이 쓰러뜨리면 승리합니다.

## 바로 하기

GitHub Pages 링크로 접속하면 설치 없이 바로 플레이됩니다.

## 수정하기

`index.html` 맨 위의 `GAME_CONFIG` 객체에서 숫자만 바꾸면
체력·속도·공격력·필살기·컴퓨터 난이도를 쉽게 조절할 수 있습니다.
