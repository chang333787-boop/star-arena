## 0. 배경 & 목적
별빛 아레나는 "세계관 + 캐릭터 6종 + 56px 타일 시스템 + 성장버프 + 모드별 맵에디터(교사 승인)"를 갖춘 플랫폼이 됐다. 이 위에 **장르 3종을 얹어 미니 게임 플랫폼으로 확장**한다. 맵에디터가 세 장르의 콘텐츠 엔진(학생 제작 → 교사 승인 → 학급 플레이)이다.

**목표(성공 기준)**
1. 세 모드 모두 봇 포함 실플레이 가능 + 에디터 제작→승인→플레이 E2E 성립.
2. 기존 4모드·캠페인·온라인 무회귀(하니스 전종 PASS).
3. 각 모드가 "3분 안에 규칙을 이해하고 웃는" 교실 파티성 확보.

**비목표(이번에 안 함)**
- 서바이버류(캠페인과 중복 — 보류) · 플랫포머 v2(세로 타워맵·벽점프·고스트 경쟁·전용 점프 스프라이트) · 사운드(에셋 대기) · 랭킹/시즌.

**대상·시나리오**
- 학생(8~13세): 쉬는시간 5~10분 플레이. 키보드/마우스/터치 3모드.
- 교사: 승인·밸런스콘솔(B)·수업 운영. 부적절 맵 걸러냄.

## 1. 별빛 술래 (술래잡기 · 규칙 id `tag`)

### 1.1 게임 플로우 (상태머신)
```
매치시작 → 술래공개(3s 카운트다운, 술래 연출) → 추격(90s) → 종료
```
- 종료 조건: ①생존자 0명(전원 감염) → **술래팀 승** ②타이머 종료 → **생존자(도망팀) 승**.
- 개인 결과: 도망자 = 생존시간 / 술래 = 감염 수. MVP = 최장 생존자(승패 무관 표기).

### 1.2 핵심 규칙
| 항목 | 값(시작값) | 비고 |
|---|---|---|
| 인원 | 3~6 (봇 채움) | 2인 이하면 시작 불가 안내 |
| 시작 술래 수 | 3~4인:1 · 5~6인:1 (감염되며 증가) | 초기 2술래는 v2 옵션 |
| 잡기 판정 | 충돌원 겹침(r+r) | 프레임당 1회, 접촉 즉시 |
| 감염 | 잡힌 도망자 → 술래팀 전환 | 연출: 플래시+링 이펙트, 팀색 즉시 변경 |
| 신규 술래 잡기 쿨 | **2.0s** | 연쇄 즉감염 방지(감염된 본인이 2초간 잡기 불가) |
| 매치 시간 | 90s | B콘솔 등록 |
| 술래 이속 | **+12%** | 도망자보다 약간 빠름 — 수풀·지형 활용 강제 |
| 전투 | Z(발사)·C(궁) **비활성** | 순수 추격 |

### 1.3 X스킬 허용표 (캐릭 개성 유지 — 이동/방해류만)
| 캐릭 | X | 술래잡기 처리 |
|---|---|---|
| 시고니 | 대시류 | ✅ 그대로 (도주/추격 스킬) |
| 눈꽃 | 서리 바닥(둔화존) | ✅ 그대로 (술래 늦추기 — 핵심 재미) |
| 럭키 | 공격형(별빛 발사) | ❌ 비활성 (아이콘 잠금 표시) |
| 달이 | 공격형(대형 비행기) | ❌ 비활성 |
| 모아 | 회복 오오라 | ❌ 비활성 (회복 무의미) |
| 별골렘 | 방패 | ❌ 비활성 |
- 비활성 캐릭은 X 독에 🔒 + "이 모드에선 사용 불가" 툴팁. (형평 논란 시 전체 비활성으로 후퇴 — 오픈 퀘스천 §5)

### 1.4 수풀 은신 (모드의 핵심 재미)
- 기존 시스템 그대로: **술래는 수풀 속 도망자 안 보임**, 근접 170px 이내면 드러남(반투명).
- 도망자끼리는 서로 보임(팀 시야). 수풀 안에서 이동 시 흔들림 이펙트(기존)로 간접 힌트 — 술래의 심리전 요소.

### 1.5 HUD·연출
- 상단: 생존자 수(❤ n명) · 타이머. 내 상태 배너: 도망자 "도망쳐!"(파랑) / 술래 "잡아라!"(빨강, 감염 직후 2s 크게).
- 술래 시각 표식: 머리 위 빨간 👹 마커 + 발밑 빨간 링(가독성 규칙과 동일 문법).
- 잡기 순간: 화면 플래시(피해 플래시 재활용) + 감염 링.

### 1.6 봇 AI
- 도망 봇: 최근접 술래 반대 벡터 + 수풀 가중치(+) + 벽 회피. 술래 300px 밖이면 수풀에 잠복.
- 술래 봇: 최근접 도망자 추적(예측 lead 0.3s). 수풀 근처에선 배회 수색.

### 1.7 에디터·엣지
- 에디터 팔레트: 벽·수풀(주력)·얼음·시작점. 거점/탑/광맥 제외. 검증: 수풀 ≥ 3덩이 권장 경고(강제 아님).
- 엣지: 온라인 술래 이탈 → 술래 0명이면 도망자 중 랜덤 재지정(3s 카운트) / 동시 다수 접촉 → 전원 감염 / 감염 순간 무적류 효과 무시(무적 아이템 없는 모드지만 방어코딩).

### 1.8 수용 기준 (AC)
- [ ] 봇 5명과 90초 매치: 감염 전파 → 승패 판정 정상(양쪽 케이스).
- [ ] 수풀 잠복 시 술래 봇이 인지 못 함(170px 밖).
- [ ] 감염 직후 2s 내 재감염 0건(하니스).
- [ ] 비활성 스킬 입력 시 발동 0 + 🔒 표시.

## 2. 별빛 칠하기 (색칠전 · 규칙 id `paint`)

### 2.1 게임 플로우
```
매치시작 → 페인팅(90s, 리스폰 무제한) → 종료 집계 연출(% 카운트업) → 승패
```
- 승리: 종료 시 **칠한 칸 수 많은 팀**. 동수 = 무승부.

### 2.2 페인트 시스템 (데이터 모델)
- `paintGrid = Uint8Array(COLS×ROWS)` — 0 무색 · 1 청 · 2 홍. 56px 셀 = 페인트 1칸(STUDENT_CELL 그리드 재활용).
- **칠 불가 마스크**: 맵 로드 시 벽·수풀·함정·구조물 footprint 칸 계산(1회) → 칠 시도 무시.
- 점수 집계: 증분 카운터(칠할 때 ±1) — 매 프레임 전수 카운트 금지.

### 2.3 칠 규칙 (시작값)
| 이벤트 | 칠 범위 |
|---|---|
| 탄 비행 | 통과 칸 1칸씩 (탄 위치 셀) |
| 탄 소멸/착탄 | 중심+십자 **5칸** |
| 적 명중 | 명중 지점 3×3 중 유효 칸 |
| 궁극기 | 폭발형: 반경 내 전 칸 / 돌진형: 경로 폭 2칸 / 다중탄: 탄별 규칙 |
- **무기 개성 = 칠 개성**(의도된 비대칭): 눈꽃(다발) = 면 도색기 / 달이(저격) = 긴 선 / 별골렘(단발 강타) = 착탄 대형 스탬프(십자 대신 3×3).
- 덧칠 = 상대색 덮어씀(무색과 동일 비용).

### 2.4 이동·전투 상호작용
| 항목 | 시작값 | 비고 |
|---|---|---|
| 자기색 위 이속 | **+10%** | B콘솔 토글(0%로 끄기 가능) |
| 적색 위 이속 | 100%(페널티 없음) | v1 단순화 |
| 킬/데스 | 점수 아님 — 리스폰 4s만 | 면적이 유일한 점수(스플래툰 정신) |
| 궁 게이지 | 명중 충전 + **칠 10칸당 +2** | 칠 행위도 보상 |

### 2.5 렌더·HUD
- 칠 렌더: 셀 반투명 팀색 틴트(청 rgba(90,160,255,.35) / 홍 rgba(255,106,106,.35)) — 밝은 중립 바닥 에셋 없이 v1. 페인트는 바닥 위·유닛 아래 레이어.
- HUD 상단: 실시간 면적 바(청|무|홍 %, 0.5s 갱신). 종료: 전체 맵 틴트 강조 + % 카운트업 연출.

### 2.6 온라인 동기화
- host 권위. 델타 전송: 변경 칸 (idx,color) 목록을 **0.2s 배치**로. 게스트 중도 입장/재동기화 = 전체 그리드 RLE 압축 스냅샷 1회.
- 패킷 상한: 배치당 최대 200칸(초과분 다음 배치) — 궁극기 대량 칠 스파이크 방어.

### 2.7 봇 AI
- 무색/적색 칸 밀도 높은 방향으로 이동+지향 사격. 적 조우: 체력 우위면 교전, 열세면 회피하며 칠. 종료 15s 전엔 대형 빈 구역 우선.

### 2.8 에디터·수용 기준
- 팔레트: 벽·수풀·함정·시작점(거점류 제외). 검증: 칠 가능 칸 비율 ≥ 55% 권장 경고.
- AC: [ ] 칠·덧칠·불가칸 판정 하니스 / [ ] 90s 봇전 승패·동수 무승부 / [ ] 델타 동기화로 게스트 그리드 일치 / [ ] 면적 바 실측 오차 0.

## 3. 별빛 점프 (플랫포머 · 별도 게임 타입 `platformer`)

### 3.1 정체성·구조
- **가로 스크롤 사이드뷰 롱맵**(마리오 계보). 탑다운과 **완전 별도 STATE·update·render 루프**(중력이 기존 모드에 새지 않게).
- 진입: 로비 "혼자 하기" → 🦘 별빛 점프 → **레벨 선택 화면**(내장 3 + 승인 레벨, 각 베스트타임 표시).

### 3.2 맵 규격·데이터
| 프리셋 | 크기(칸) | 용도 |
|---|---|---|
| 연습 | 33×12 | 1.5화면 — 첫 제작용 |
| 스탠다드 | 66×12 | 3화면 |
| 롱 | 110×14 | 5화면 — 고급 |
- 데이터: `{ type:"platformer", size:[cols,rows], rows:["문자열"...], meta:{name,author,bestTime} }` — 미러 없음.
- **타일 legend**: `#` 솔리드 · `-` 원웨이 · `S` 가시 · `J` 스프링 · `M` 순찰몬스터 · `o` 별조각 · `F` 체크포인트 깃발 · `G` 골 · `*` 시작점 · `.` 공기.
- **낙사**: 플레이어 y > 맵 하단 +2칸 → 사망.

### 3.3 물리 스펙 (전부 상수화 — B콘솔 등록)
| 상수 | 시작값 | 설명 |
|---|---|---|
| GRAVITY | 2200 px/s² | 상승 중 점프 홀드 시 ×0.6 (가변 점프) |
| JUMP_V | -820 px/s | 점프 초속 |
| JUMP_CUT | ×0.45 | 버튼 릴리즈 시 vy 컷(탭 점프 ≈1.8칸, 홀드 ≈3.2칸) |
| MOVE_SPD | 220 px/s | 지상 최고속 |
| ACC_GROUND / ACC_AIR | 1800 / 1100 | 가속(에어컨트롤) |
| FRICTION_GROUND / AIR | 2200 / 200 | 감속 |
| TERMINAL_VY | 900 | 종단 낙하속도 |
| COYOTE / BUFFER | 0.10s / 0.10s | 코요테타임·점프버퍼 |
| SPRING_V | -1300 | 스프링 발사 |
| STOMP_BOUNCE | -500 | 밟기 후 바운스 |
- 충돌: 56px 타일 AABB, 축분리(x 먼저, y 나중). 머리 박기 = vy 0. **원웨이(`-`)**: 하강 중 + 발이 타일 상단 위→아래 교차 시만 착지, `↓+점프` 0.25s 통과 하강.

### 3.4 적·오브젝트
- **순찰 몬스터(M)**: 좌우 60px/s 왕복, 벽·낭떠러지에서 반전(소형 몬스터 에셋 재활용).
  - **밟기 판정**: 플레이어 하강 중(vy>0) + 발이 몬스터 상단 25% 이내 → 처치+바운스. 그 외 접촉 → 사망(깃발 리스폰).
- 가시(S): 접촉 즉사(깃발 리스폰). 스프링(J): 접촉 시 SPRING_V. 별조각(o): 수집(레벨당 수집률 기록). 깃발(F): 통과 시 활성(체크포인트 갱신, 색 변화). 골(G): 도달 = 클리어.
- 사망 = 무제한, 즉시 깃발 리스폰(교실 친화 — 하트 시스템 없음, 오픈 퀘스천 §5). 기록 = 클리어 타임 + 별조각 n/총.

### 3.5 카메라 (사이드뷰 전용)
| 항목 | 값 |
|---|---|
| x 추적 | 플레이어 중심 + 진행방향 lookahead **+80px**, 데드존 60px |
| y 추적 | **착지 y 기준 스냅 + lerp 8/s** — 점프 중 y 고정(출렁임=멀미 방지), 착지 층 바뀔 때만 이동 |
| 경계 | 맵 밖 안 보이게 클램프 |
| 컬링 | 뷰포트 ±2칸 밖 타일·엔티티 스킵 |

### 3.6 조작 (3모드 연동)
- 키보드: ←→ 이동 · **Z/Space** 점프(가변) · ↓+점프 원웨이 하강 · R 재시작(깃발부터) · Esc 나가기.
- 마우스 모드: 동일 키(마우스 미사용 — 조준 개념 없음). 터치: 좌하단 ◀▶ 버튼 + 우하단 대형 점프 버튼.

### 3.7 내장 레벨 3종 (레벨 디자인 가이드)
1. **첫 점프**(연습 33×12): 평지→단차→2칸 구멍→원웨이 계단→골. 가시·몬스터 없음. 학습 곡선: 이동→점프→가변점프.
2. **별빛 언덕**(스탠다드 66×12): 몬스터 2·가시 구간·스프링 1·깃발 1·별조각 5. 밟기와 회피 학습.
3. **구름 꼭대기**(스탠다드 66×12): 원웨이 중심 수직 구간·스프링 연쇄·깃발 2·별조각 7. 정밀 점프.

### 3.8 플랫포머 에디터
- 모드 "플랫포머" 추가(기존 에디터 프레임). **대칭 OFF**.
- **스크롤 편집**: Shift+←→ = 화면 단위 이동, 상단 **미니맵**(전체 축소 + 현재 뷰포트 표시), Tab = 전체보기 토글.
