게임을 19개 축(아래)으로 전수 검토한다. 각 축마다: 현 상태 코드/플레이로 확인 → 문제 발견 → **작고 명확하면 바로 수정**, **크면(구조) 기록+권고만**. 별도로 **하단 스킬 HUD를 게임식 아이콘 UI로 재설계**(이번 실구현).
근거: `게임평가_상용화갭_20260705.md`(이미 한 분석 — 중복 재분석 말고 그 위에 축별로 정리·보강) · `게임분석_세부10건_20260705.md`.

원칙(중요):
- **작고 명확한 버그/폴리시 = 지금 수정.** 큰 구조작업(사운드 엔진·무빙샷 조준개편·마우스터치·서버권위·맵 클라우드화)은 **확인+기록+권고만, 이번에 구현 금지**(사용자 결정 필요). 애매하면 기록으로 남기고 넘어가기.
- 추측 금지 — 각 축 판정은 코드/화면에서 확인한 사실로.

■ 공통 금지: index.html↔star_arena.html 바이트 동일 깨기 / harness 전역 심볼 변경 / 캠페인·온라인·saveProfile 파손 / 대규모 리팩터링(단일파일 분리 등).
■ 공통 허용: 렌더·HUD·밸런스·검증·수정, 새 상수/함수, 하니스/테스트/문서.
■ 중단조건: 회귀로 캠페인 클리어 불가 / 특정 축 수정이 과대 → 미수정 기록하고 다음 축.

━━━━━━━━━━━━━━━━ ★ 우선 구현: 하단 스킬 HUD 재설계 ━━━━━━━━━━━━━━━━
현재 하단이 텍스트("C 궁극기·별똥별·80%")라 애들이 직관적으로 못 읽음 → **게임식 아이콘 인디케이터**로.
- **Z (기본공격/탄창)**: 원형 인디케이터. 탄창 있는 캐릭 = **탄창 수만큼 나눈 링**(쏘면 한 조각씩 소진, 장전 시 채움). 무한탄(럭키 등) = ∞ 표시 or 발사속도 펄스.
- **X (특수기술/쿨타임)**: **원형 쿨타임 스윕** — 쓰면 시계방향으로 차오르고, 다 차면(준비) 밝게 빛남.
- **C (궁극기/게이지)**: **노랑·골드 게이지 차오름** — 명중으로 충전, 100%면 반짝/펄스.
- 각 항목: 스킬 아이콘(있으면 활용, 없으면 도형/글자) + 키(Z/X/C) + 이름 + **"준비됨" 시각신호**. 글자 의존 최소화.
- 캔버스 렌더(기존 HUD 함수 확장). 하니스 fallback 유지. index/star_arena 동일 반영.

━━━━━━━━━━━━━━━━ 19개 축 검토 (각: 확인 → 수정 or 기록) ━━━━━━━━━━━━━━━━
[기능]
1. 모드 승패조건 — 4모드(섬멸·점령전·수정부수기·별모으기) 승리/무승부/시간종료 실제 작동. 버그 시 수정.
2. 스킬 정합성 — 캐릭 6종 Z/X/C/패시브 의도대로 + 무한콤보·오작동 없나(럭키 C 상한·모아 패시브 기수정분 회귀 확인).
3. 온라인 동기화 — 호스트/게스트 일치·끊김·빈자리봇. (서버권위 C1은 구조=기록만.)
4. 에디터→승인→플레이 — 빈맵·검증실패·중복이름·거절 엣지. 작은 버그 수정. (클라우드화·삭제도구 D2·D3=구조=기록.)
5. 저장/로드 무결성 — 프로필·에디터맵·승인맵 localStorage 손상·버전 마이그레이션. 방어코드 없으면 추가.
[최적화]
6. 프레임레이트 — 크롬북 6인 난전. (코드감사 기최적화분 유지 확인.)
7. 메모리 누수 — 여러 판 후 effects·bullets·listener 잔존. 발견 시 정리.
8. 네트워크 부하 — DB 쓰기(기수정 96%↓ 확인). 추가 낭비 있나.
9. 에셋 로딩 — 404 폴백·캐시(?v=). 깨진 참조 있나.
10. 발열/배터리 — 유휴 시 불필요 렌더/루프(로비에서도 rAF 도는지 등). 발견 시 절감.
[디자인/UX]
11. **난전 가독성(확인된 문제)** — "적이랑 헷갈림". 팀색 대비 강화·적 외곽선/발밑 마커·나 강조 등으로 **아군/적/나 즉시 구분** 되게 수정.
13. 정보구조 — 로비·에디터·교사패널 네비 명확성. (OS prompt 남용은 폴리시, 여력되면 인게임 입력으로.)
14. 시각 일관성 — 새 지형·캐릭(3화풍)·UI·HUD 톤 통일. 튀는 것 기록/정리.
15. 피드백감 — 타격·처치·승패·버프획득 시청각. (사운드=구조=기록. 시각 피드백 약하면 보강.)
[밸런스]
16. 캐릭터 밸런스 — 6종 승률/OP·약캐(별골렘 버프 후 체감 포함). 하니스/시뮬로 편향 확인, 시작값 조정 여지 기록.
17. 모드 밸런스 — 매치길이·역전·스노우볼(포탑450·별60초 등). 극단값 기록.
18. 봇 난이도 — 난이도별 체감차. 방치승리 완화 여지 기록.
[접근성/교실]
19. 접근성 — 색맹·저시력·텍스트크기·키보드편의. (마우스터치=구조=기록.) 색맹 대비 등 작은 개선.
20. 교실 운영 — 교사 맵관리·밸런스콘솔(기구현) 편의·부적절 콘텐츠 방지(악성배치 검증)·개인정보 최소화.
