━━━━━━━━━━━━ ① 술래잡기 "별빛 술래" (제일 쌈 — 파티 모드) ━━━━━━━━━━━━
규칙(감염형):
- 시작 시 1명 랜덤 술래(연출: 3초 카운트+술래 표시). **잡기 = 접촉(충돌원 겹침)** → 잡힌 사람도 술래팀 합류(감염).
- **마지막 생존자 승리** / 제한 90초 내 전원 감염 시 술래팀 승.
- **전투 무효**: Z/궁 비활성(또는 미적용). 이동 + X 기동기만(캐릭별 개성 유지 — 시고니 대시 등. 공격성 X스킬은 효과 무해화 or 비활성, 캐릭별 판단해 기록).
- 밸런스 시작값: 술래 이속 +12% · 감염 직후 2초 무적(연쇄 즉감염 방지) · 도망자는 미니맵에 술래 방향 힌트 없음.
- **수풀 은신 활용(핵심 재미)**: 기존 은신 시스템 그대로 — 술래는 수풀 속 도망자를 못 봄(근접 시 드러남). 숨는 재미 공짜.
- 규칙 셀렉터에 `tag` 추가(빠른대전+온라인). 봇 지원(도망 AI = 술래 반대방향+수풀 선호 / 술래 AI = 최근접 추적).
- HUD: 남은 생존자 수 · 타이머 · 내가 술래인지 크게 표시.
- 에디터 연동: 모드 팔레트 = 벽·수풀(많이)·얼음/부스터류·시작점. 거점/탑 등 무관 요소 제외.

━━━━━━━━━━━━ ② 색칠전 "별빛 칠하기" (스플래툰식) ━━━━━━━━━━━━
규칙:
- 팀 2색(청/홍). **탄이 바닥에 닿거나 지나간 칸을 팀색으로 칠함**(56px 셀 단위 — STUDENT_CELL 그리드 재활용). 명중한 적 주변도 소량 칠.
- 90초 종료 시 **칠한 면적 % 많은 팀 승**. 킬은 리스폰(4초)만 — 점수는 오직 면적(스플래툰 정신).
- paint 상태 = `Uint8Array(COLS×ROWS)`(0=무·1=청·2=홍) — 온라인은 host 권위 + 델타 동기화(변경 칸만 전송, 패킷 절제).
- 렌더: 칠해진 칸을 반투명 팀색 틴트(밝은 중립 바닥 에셋 없이 코드 틴트로 v1). 벽/수풀 칸은 칠 불가.
- 시작값: 자기색 위 이속 +10%(선택 — 과하면 끄기), 궁 게이지는 칠한 칸수로도 소량 충전.
- HUD: 상단 실시간 면적 게이지 바(청/홍 %).
- 규칙 셀렉터 `paint` 추가. 봇 = 빈 칸 방향 이동+발사.
- 에디터 연동: 팔레트 = 벽·수풀·함정·시작점(거점류 제외). 칠 가능 영역이 넓은 개방형 맵 권장 안내.

━━━━━━━━━━━━ ③ 플랫포머 "별빛 점프" — 가로 스크롤 롱맵 (빡센 설계) ━━━━━━━━━━━━
정체성: **가로 스크롤 긴 맵**(마리오 계보). 싱글스크린 아님 — 카메라 포함 정식 설계.

3-1 맵 구조:
- 크기 프리셋: **연습 33×12 · 스탠다드 66×12 · 롱 110×14** (세로형 22×48은 v2 기록).
- **체크포인트 깃발**(요소) — 사망 시 마지막 깃발에서 리스폰. **낙사**: 최하단 아래로 떨어지면 사망(구멍 = 레벨 요소).
- 데이터: 기존 문자열 그리드 rows 그대로, cols만 확장(미러 없음).

3-2 물리(신규 코어 — 손맛 디테일이 본체):
- 중력·종단속도 · **가변 점프**(홀드 시간 비례) · **코요테타임 0.1s + 점프버퍼 0.1s** · 머리 박기(천장) · 공중 가속/감속(에어컨트롤).
- 타일 AABB(56px): 솔리드 플랫폼 + **원웨이 플랫폼**(밑에서 통과·위에서 착지·↓+점프 하강) — 레벨디자인 핵심 요소.
- 무빙 플랫폼(좌우/상하 왕복, 캐릭터 운반 판정) — 여력 되면 v1.5, 안 되면 기록.
- 시작값: 이속 220 · 점프높이 ~3.2칸(홀드) / ~1.8칸(탭) · 중력 2200 · 종단 900. 전부 상수화(콘솔 튜닝 가능).

3-3 카메라(사이드뷰 전용 — 탑다운과 별개):
- x축: 플레이어 추적 + 진행방향 lookahead(+80px) + 데드존(60px). 맵 경계 클램프.
- y축: **착지 기준 스냅+스무딩**(점프마다 출렁임 금지 — 멀미 방지). 세로 이동 클 때만 부드럽게 따라감.
- 렌더 컬링: 뷰포트 밖 타일/엔티티 스킵(롱맵 프레임 방어).

3-4 적·요소(재활용 중심):
- **순찰 몬스터**(좌우 왕복, 낭떠러지서 회전 — 몬스터 에셋 재활용) · **밟으면 처치**(위에서 밟기 판정, 닿으면 피해→깃발 리스폰).
- 스프링(고점프) · 별조각(수집, star pickup) · 골 별(star_vein) · 가시(spike — 밟으면 사망) · 체크포인트 깃발(도형 v1 가능).
- 클리어 = 골 도달. 기록: 클리어 타임 + 별조각 수집률(개인 베스트 저장).

3-5 조작(3모드 연동):
- 키보드: ←→ 이동 · Z/Space 가변점프 · ↓+점프 원웨이 하강.
- 마우스 모드에서도 동작(이동/점프 동일 키). 터치: 좌우 버튼 + 점프 버튼(우측 큰 버튼).

3-6 플랫포머 에디터(긴 맵 대응):
- 모드 "플랫포머" 추가. **에디터 카메라**(좌우 스크롤 편집) + **전체보기/미니맵** 토글.
- 팔레트: 솔리드 플랫폼 · 원웨이 플랫폼 · 가시 · 스프링 · 순찰몬스터 · 별조각 · 체크포인트 깃발 · 골 · 시작점. **대칭 OFF**.
- 검증: 시작점·골 각 1 필수 + **제출 전 "직접 클리어" 필수**(에디터 내 테스트플레이로 클리어해야 제출 활성 — 도달성 보장).
- 교사 승인 흐름 동일 → 승인 레벨 = "별빛 점프" 레벨 목록.

3-7 엔진 분리·안전:
- 탑다운 update/render와 **완전 별도 STATE·루프**(중력이 기존 모드에 새지 않게). ARENA는 뷰포트로 재해석(플랫포머 한정).
- harness 전역 불변. 신설 하니스: 점프높이(탭/홀드)·코요테·원웨이 통과/착지·낙사·체크포인트 리스폰·밟기 판정.
- 진입: 로비 "혼자 하기" → **🦘 별빛 점프**(내장 레벨 2~3 + 승인 레벨). 온라인/고스트 경쟁은 v2 기록.
