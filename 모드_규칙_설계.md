# 모드(규칙) 설계 — 별빛 아레나 PvP 오브젝트 모드

> 작성: 2026-07-05 · 용도: "규칙" 셀렉터에 들어갈 각 모드의 게임플레이 설계. 구현 전 규칙 확정용.
> 3층 모델의 ②규칙 층. 각 모드 = 규칙 셀렉터 한 줄. 색칠전은 시즌2.
> 원칙: 표절 회피 = 메커닉 섞기 + 별빛 세계관 리스킨. 교실용 = 짧고 명확하고 읽힘.

## 공통 프레임 (전 오브젝트 모드)
- 팀: **3v3** (빈자리 봇 채움 · 오프라인 봇전/온라인 대인). 1v1도 가능.
- **캐릭터전**(몬스터 아님). 몬스터는 캠페인 전용.
- 매치 길이: **2~4분** (교실 회전).
- **리스폰 있음** (죽어도 잠깐 뒤 부활 — 오브젝트 모드는 압박 지속. 섬멸전과 차별점).
- **강화 3택1** 매치 중 성장 유지 (이 게임 시그니처, 모드마다 재활용).
- 캐릭 Z도구/X특수/C궁극 그대로.

---

## 1. 🏰 수정부수기 (별성 공성전)
**세계관 이름 후보**: 별성 공성전 · 수정 깨기 · 별핵 부수기

- **승리 조건**: 상대 팀 **별 수정탑**을 먼저 파괴. (시간 종료 시 탑 HP 낮은 팀 패)
- **핵심 루프**:
  - 각 팀 진영에 별 수정탑 1개(HP 높음). **탑이 근처 적을 자동 공격(포탑)**.
  - 정면 돌격 = 포탑에 맞음 → **파괴/건설 가능한 벽(쿠션벽)**으로 진입로·엄폐 만들며 밀기.
  - 공격/방어 동시에 신경 (우리 탑도 지켜야).
- **재미**: 공성 vs 수성 밸런스, 벽으로 진형 짜기, "때려부수기" 쾌감.
- **재활용도 ★★★ (제일 쌈)**: 캠페인 코어 방어(방어 수정 75초) + 구조물 HP + 파괴 벽 이미 있음 → 대칭 PvP화 + **포탑 자동공격 로직만 신규**.
- **필요 존 타일**: 별 수정탑(청/홍) · 건설 벽.
- **밸런스 노브(제안)**: 탑 HP, 포탑 데미지·사거리, 벽 개수/HP. 양팀 완전 대칭.

## 2. ⭐ 별모으기 (별 사냥)
**세계관 이름 후보**: 별 사냥 · 별빛 모으기 · 별 줍기

- **승리 조건**: 팀 합계 **별 10개** 모은 뒤 **15초 카운트다운** 버티면 승. (뺏겨 10개 밑이면 카운트 리셋)
- **핵심 루프**:
  - 중앙 **별광맥**에서 주기적으로 별이 솟음 → 주워서 보유.
  - **죽으면 들고 있던 별을 그 자리에 떨어뜨림**(뺏김). → 많이 든 사람이 표적 = 긴장.
  - 팀 합계 10 도달 → 카운트다운 → 버티면 승.
- **재미**: "많이 든 사람 지키기 vs 노리기", 별 든 채 도망 스릴. **별 아레나 = 별 모으기, 테마 완벽.**
- **재활용도 ★★**: 픽업 시스템(heal) 모델 + 별 pickup 에셋. 별 보유/드랍/카운트 신규.
- **필요 존 타일**: 중앙 별광맥 · 별 pickup.
- **밸런스 노브(제안)**: 별 스폰 속도, 목표 개수(10), 카운트다운(15s).

## 3. ✨ 핫존 (빛나는 땅)
**세계관 이름 후보**: 빛나는 땅 · 별자리 자리 · 별빛 지대

- **승리 조건**: **거점 점유 게이지 100%** 먼저. (시간 종료 시 게이지 높은 팀)
- **핵심 루프**:
  - 맵 중앙(또는 2곳)에 거점 링. 팀원이 안에 있으면 게이지 참.
  - **양팀 다 있으면 정지(경합)** → 밀어내야 함.
  - 리스폰 있음, 좁은 곳 난전.
- **재미**: 위치 선점, 좁은 구역 교전 집중. 브롤 핫존 익숙함(애들 언급) + 별빛 리스킨.
- **재활용도 ★★**: 영역 판정 로직(X스킬 얼음 영역 등) 재활용. 거점 게이지 신규.
- **필요 존 타일**: 거점 데칼(중립→팀색).
- **밸런스 노브(제안)**: 거점 1개 or 2개, 게이지 속도, 경합 규칙.

## 구현 우선순위 (재활용도 순)
1. **수정부수기** — 코어/벽 이미 있어 제일 쌈. 포탑 공격만.
2. **핫존** — 영역 판정 재활용, 거점 게이지 신규. 중간.
3. **별모으기** — 픽업 재활용, 보유/드랍/카운트 신규. 중간.
(섬멸전 = 이미 있음. 색칠 = 시즌2, 시스템 신규라 제일 무거움.)

## 미결(사용자 확정 필요)
- 세계관 이름 최종 선택(각 모드).
- 수정부수기: 탑 1개 vs 추가 포탑들. 핫존: 거점 1 vs 2. 별모으기: 목표 개수.
- 리스폰 규칙 세부(부활 시간, 목숨 제한 여부).
- 강화 3택1을 PvP 모드에도 그대로 넣을지(밸런스 영향).
