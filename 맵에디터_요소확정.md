```js
{ id:"student_xxx", name:"맵이름", author:"학생이름", size:[22,10]|[36,16],
  mirror:"lr"|"rot180",             // 좌우 거울 | 180° 회전
  cells:"....W..B..." ,             // 왼쪽 절반만 저장(행 우선 문자열) — 나머지는 로드 시 미러
```

- 검증(로드 공통): ★ 1개 필수 · 스폰↔거점/상대 진영 연결성(플러드필) · 벽 ≤ 25% · 풀숲 ≤ 15%.
