설정(⚙)에 **"조작 방식"** 옵션: 키보드 / 마우스 / 터치(태블릿). 프로필 저장.

3-0 입력 추상화(선행): 세 입력 소스를 동일한 **의도**로 변환 → 게임로직은 의도만 소비.
   의도 = { moveVec(dx,dy), aimAngle, fireHeld, useX, useC, reload }.
   ※ 온라인 입력 전송(emptyInput/입력 패킷)은 **의도 기준으로 통일** — 소스 달라도 전송 포맷 동일(동기화 안전).

3-1 키보드(현재 유지): 방향키 이동, aim=이동방향, Z 발사·X·C·R·P. **동작 변경 없음**(기본값).

3-2 마우스 모드(무빙샷 해결):
   - 이동 = **WASD**, 조준 = **마우스 커서**(이동과 분리 → 카이팅 가능).
   - **좌클릭 = 발사**(홀드 연사), **우클릭 = X 특수**, **스페이스 = C 궁**, R 장전.
   - 캔버스 좌표 변환(스케일·오프셋 역산)으로 커서→월드 조준각.

3-3 터치 모드(태블릿·브롤식):
   - **좌 가상 조이스틱 = 이동**, **우 가상 조이스틱 = 조준+발사**(놓으면 발사/브롤식).
   - 화면 **스킬 버튼**(X·C·장전) — 하단 스킬독을 탭 가능하게 겸용.
   - touchstart/move/end 처리, 멀티터치. 캔버스 좌표 변환 공용.

3-4 HUD/안내: 모드별 하단 조작 안내 문구 교체. 스킬독은 3모드 공용.
