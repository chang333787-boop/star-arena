## ★ 우선 구현 — 하단 스킬 HUD 재설계 (완료)

텍스트 HUD("C 궁극기·80%") → **게임식 아이콘 독** 3원형, 로컬(drawHUD)·온라인 전 모드(drawCombatStrip)가 같은 `drawSkillDock()` 공용.

| 슬롯 | 표현 |
|---|---|
| **Z** | **탄창 수만큼 분할된 링** — 쏘면 조각 소진, 2발 이하 빨강. 장전 = 차오르는 링(게스트는 진행도 미수신 → 회전 스피너). 무한탄(럭키) = 금색 링 + ∞. 중앙엔 무기 발사체 아이콘(에셋), 없으면 남은 발수 |
| **X** | **원형 쿨타임 스윕** — 준비된 만큼 시계방향 부채꼴 + 남은 초 표시, 준비되면 캐릭터별 글리프(✶◎▲❄♥◆)가 초록 펄스로 빛남 |
| **C** | **골드 게이지 링** — 명중 충전으로 차오르고 % 표시, 100%면 ★ + 외곽 펄스 링 |
| 공통 | 스킬 이름 라벨(폭 맞춰 자름 — measureText) + **키캡(Z/X/C)**, 준비되면 키캡이 노랗게 점등. 글자 의존 최소화 |

브라우저 실측: 럭키(∞)·달이(5분할 링+발사체 아이콘·쿨 스윕 4.2초·30% 게이지·★ 풀차지) 스크린샷 확인.
