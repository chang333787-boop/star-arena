# 별빛 아레나 — 냉철 평가 & 상용화 갭 분석 (2026-07-05, v1.30 기준)

관점: ①10년차 게이머의 플레이테스트 ②상용 서비스 PM의 기본기 점검.
방법: 코드 전수 확인(기능 유무는 전부 코드로 검증) + 실플레이 + 전 화면 답사. 추측 아님 — 각 항목은 코드/화면에서 확인한 사실.

## 총평 (한 줄)

> **"수업용 프로젝트로는 A급, 상용 잣대로는 아직 데모(pre-alpha)다. 코어 루프(전투→성장→보상)는 뼈대가 있는데, '소리·조준·마우스'라는 3대 기본기가 통째로 없어서 신규 유저를 5분 이상 붙잡을 힘이 없다."**

10년차 게이머 첫 15분 시나리오: 게임을 켠다 → 소리가 하나도 안 난다(버그인 줄 앎) → 화살표로 움직여 보는데 **뒤로 빠지면서 쏠 수가 없다**(조준=이동방향) → 마우스를 클릭해 본다, 아무 반응 없음 → 봇전 이기고 골드 150G 획득 → 상점이 "준비 중" → **이 골드 어디에 쓰지?** → 이탈.

## A. 코어 게임플레이 — 10년차 게이머의 냉평

| # | 문제 | 근거(코드/실측) | 심각도 |
|---|---|---|---|
| A1 | **무빙샷(카이팅) 불가** — 조준이 이동 방향에 종속. 후퇴하며 사격, 옆으로 빠지며 견제가 원천 불가능. 브롤러 장르의 실력 표현 절반이 사라짐. 실력 격차 = 거리 재기뿐 | `updatePlayer`: `facing=Math.atan2(my,mx)`(이동 방향), 발사는 facing 고정. 자동 조준도 없음 | ★★★ |
| A2 | **완전 무음** — 사운드 코드 0줄(Audio/AudioContext/bgm 검색 0건). 타격감·피격 경고·궁 차징의 절반은 소리다. 상용 기준 실격 사유 | grep 0건 | ★★★ |
| A3 | **골드가 쓸 곳이 없다** — 상점 비활성(SHOP_ENABLED=false), 골드 차감 코드 0건. 이기면 +150G인데 소비처 0 → 보상 루프가 끊긴 게임 | grep `gold-=` 0건 | ★★★ |
| A4 | **방치 승리 가능** — 봇전에서 손 놓고 있어도 아군 봇이 15:1로 이겨줌(실측). 방치로 골드 수급 → 이코노미와 난이도 설계 둘 다 구멍 | 실플레이 스크린샷(준 피해 0, 승리 +150G) | ★★☆ |
| A5 | **계정 성장이 없다** — level/exp 필드는 저장되는데 **획득 로직이 0건**(죽은 데이터). 매치가 끝나면 남는 게 골드뿐인데 그 골드도 A3 | grep `exp+=`/`levelUp` 0건 | ★★☆ |
| A6 | 캐릭터 6종 + 무기 고정 → 빌드 다양성 없음. 성장 오브는 매치 내 한정이라 "내 계정만의 것"이 없음 | STUDENT_CHARACTERS 6종 | ★★☆ |
| A7 | 봇 AI가 밋밋함 — 거리 유지+랜덤 스트레이프+확률 스킬이 전부. '어려움'도 수치만 세지 패턴이 없어 배울 게 없음 | updateAI/tBotUpdate 구조 | ★☆☆ |

## E. 서비스 기본기(상용 전환 시 필수인데 없는 것)

- **운영**: 공지/점검 모드/강제 업데이트 없음. 문제 생겨도 유저에게 알릴 채널이 게임 안에 없음.
- **데이터**: 플레이 분석(어떤 모드·캐릭이 인기인지)·에러 리포팅 0줄 — 뭘 개선할지 측정 불가.
- **모더레이션**: 닉네임 금칙어/중복 검사 없음(길이만), 신고 기능 없음. UGC(맵 이름·제작자명)도 무필터.
- **계정 보안**: PIN 4자리 + 클라이언트 해시. 비밀번호 찾기 없음(교사 초기화뿐 — 교실용으론 적절, 상용은 불가).
- **법적 고지**: 이용약관·개인정보 처리방침 없음(상용 시 필수).
- **리텐션 장치**: 일일 미션·도전과제·시즌·랭킹 0 — "내일 또 켤 이유"가 설계에 없음.
- **엔지니어링**: 단일 7,000줄 HTML — 지금까진 굴러가지만 협업·코드리뷰·부분 배포가 불가능한 구조. 상용 전 모듈 분리 필요.
- 잔 결함: 맵공방 모드 버튼 설명 잘림("…카운트다운을 지"), 파비콘 404.

## 우선순위 로드맵 제안

**P0 — "게임이 게임이 되는" 필수 4종 (다음 스프린트 강력 권장)**
1. **사운드**: 발사·피격·처치·궁·승패 10종 내외 효과음 + 로비 BGM 1곡이면 체감 급상승. (Web Audio, 에셋만 있으면 하루)
2. **조준 개선**: 최소안 = 근접 자동조준(브롤스타즈 탭샷식) / 정석 = 마우스 조준 + 이동 분리(무빙샷). A1이 풀려야 실력 게임이 됨.
3. **마우스 지원**: 메뉴 클릭 + 조준 겸용. (터치는 그 다음)
4. **골드 소비처**: 상점 재개(스킨/이모지 같은 밸런스 무관 치장템이 안전).

**P1 — 멀티·UGC 신뢰성**
5. 맵공방 클라우드화(Firebase에 승인 맵 저장 → 학급 전체 공유) + **운영자 관리(삭제·비활성·승인취소)** — D2·D3 동시 해결.
6. 대형 맵: 카메라(줌아웃/팔로우) 작업해서 탑재하거나, 탑재 전까지 제작 옵션 숨김.
7. 에디터: 초안 자동저장 + 실행취소 + 제출 전 테스트 플레이.
8. 온라인: 공개 방 목록(같은 학급 uid 범위) + 재접속 + meta에 프로토콜 버전.

**P2 — 상용 기본기**
9. 서버 권위(최소한 Cloud Functions로 결과 검증) + DB 보안 규칙 정비.
10. 분석·에러 수집, 닉네임/맵이름 금칙어, 튜토리얼(첫 부팅 90초 봇전 가이드).
11. 리텐션: 일일 미션 3개 + 계정 레벨(경험치 획득 로직 연결 — 필드는 이미 있음).

**P3 — 확장**
12. 랭킹/시즌, 관전, 캐릭터 7번째+, 모바일 터치 최적화.
