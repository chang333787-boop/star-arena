## A. 배치 구성 (구현 순서)

### POLISH-1 — 게임성 코어 (P0 · 한 세트로 동시 배포 · 코드 ~330줄+데이터 26엔트리)
서로 밸런스 의존(밀도↔리스폰↔이름표↔어그로)이라 쪼개면 안 됨.

**1. 월드 텍스트 역스케일 + 몬스터 이름표 (피드백 ③ · 모든 표시의 선행 조건)**
- 공통 헬퍼 `rpgUiFont(px)` = px/RPG.cam.s (한도 12~27) — 데미지 숫자·NPC 이름·HP바·말풍선·❗ 전부 적용. 암색 외곽선(strokeText #0a081a, 3/s)으로 초지 위 가독+색약 대응.
- 이름표: **발밑**(y=mob.r+6, NPC는 위=아군 / 아래=적 이분법), **상시 표시**(이름 학습이 목적 — 단 비어그로·원거리는 알파 0.5), pill 배경(rgba(0,0,0,0.45)). 색: tier1 흰 / tier2 연보라 / 정예 노랑 "★" / 보스 금색+이모지 접두. 도감 미등록이어도 실명(이름표=인지, 도감=기록).
- 첫 조우 시 "📖 새 몬스터 발견: ○○" 토스트 + flags.met_id.

**2. 몬스터 밀도 증설 (피드백 ① · 데이터 26엔트리, 신규 종 없음)**
- 확정 수: 들판 8→**15**(30칸/마리) · 숲길 7→**21**(39) · den1 3→**6** · den2 2→**4**(수집방 컨셉 유지) · den3 보스 단독 유지.
- 좌표는 패널 확정안 사용(스폰 렌즈 산출물 — 부록 X 좌표표). 배치 규칙: 비solid만·진입 포탈 6칸(실내 4칸) 밖·주동선 차선 금지·스폰 간 3칸 이상(어그로 연쇄≤3)·같은 종 2~3 무리 배치 2곳(밀도 체감).
- 존 구성: 숲길 강 서=부슬이(9) / 동=부엉이(8) / 요새 문 앞=그늘 조각(4, 경비 관문 연출). 전 스폰(고정·레이드·소환) `rpgFindFree` 스냅 경유(벽 속 생성 방지 B-11 동시 해결).
- 진행 곡선: 두 필드 1회전=439exp→Lv7 부근, 리스폰 1~2사이클로 Lv9 — 기존 커브 재조정 불필요(오히려 의도 페이스에 수렴).

**3. 몬스터 영속 + 시간 리스폰 (피드백 ② — 도어파밍 봉인 겸용 · ~45줄)**
- 시계: **플레이 시간(sv.playSec)** 기준(실시간 아님 — 차시 간격·탭 닫힘에 공정, 시계 조작 무관). 필드에서만 누적.
- 저장: `sv.resp={"mapId#스폰idx": killedAtPlaySec}` 1필드(<2KB·자가 청소). 세이브 v1→2 마이그레이션(rpgMergeSave 백필).
- 쿨다운: 티어1 **90s** / 티어2 **150s** / 그늘 조각 **240s**(RESPAWN_OVR) / 보스 mq07 게이트 유지. 맵 로드 시점만 판정(눈앞 팝인 없음 — 트리클은 P2). **취침 시 전량 부활**(sv.resp={}) — "자야 내일 거리가 생긴다"로 ④와 시너지.
- **pity 드랍(필수 동반)**: 활성 collect/kill 퀘스트 대상 드랍 확률 2배 + 3킬 연속 미드랍 시 확정 — mq03/sq01이 리셋 파밍 전제 수치였던 것 보정.

**4. 워프 직후 기습 방지 (~3줄)**: `warpLock` 동안 어그로 340→120 (den1 정예 224px·den2 부슬이 286px 즉시 어그로 실측 해결).

**5. 전투 게임필 6종 + fx 인프라 (juice 가성비 톱6 · ~65줄)**
- fx 배열 파티클 겸용 확장(kind:text/dot/star·중력·상한 120) → ①근접 스윙 궤적(현재 근접 공격이 화면에 안 보임!) ②히트스톱(명중 0.03/기습 0.05/처치 0.08/보스 0.22 — Math.max 병합) ③넉백(140/기습 200, 감쇠·벽 침투 불가) ④피격 스쿼시+화이트 플래시(현행 "맞으면 흐려지는" 역효과 교정, ctx.filter 폴백) ⑤드랍 자석(110px 내 가속 흡인 — 4줄 최고 가성비) ⑥유휴 호흡 애니(3줄로 8종 전부 생물화). 사망 ✨텍스트→별 파티클 6개(보스 18개).

**6. UX 버그급 3종 (~105줄)**
- 가방 스크롤+표시 정렬(소모품<씨앗<작물<재료<무기<방어구, 뷰→원본 인덱스 매핑 필수) — 9번째 아이템부터 커서가 화면 밖(실플레이 15종 도달).
- 보상 배너: 퀘스트 완료 시 토스트 연속 덮어쓰기로 "+100G" 유실 → 상단 슬라이드 배너 1장에 합산(레벨업·목표 달성도 배너로 승격). 배너 활성 중 토스트 y 하향 1줄.
- **별샘 워프망**: 우물(마을)·모닥불(들판)·요새 1번 방 입구 — 최초 접촉 해금 → choice UI로 이동. 사망 복귀 40~50초 문제 해결(부활 지점 옆이 우물). den1 워프는 mq07 게이트 검사 동일 적용(우회 방지).

**7. 터치 완결성 3종 (~60줄 · 태블릿 교실이라 사실상 P0)**: 나가기 칩(현재 **터치로 RPG 종료 불가**) · 사탕 버튼+잔량 칩(현재 전투 중 사용 불가) · 상점/가방 터치 스크롤.

**8. 사탕 경제 (~30줄)**: CANDY_MAX 5 시행(기보유 초과분 몰수 금지·추가 획득만 캡) + 사용 쿨 2.0s + HUD 🍬 칩. regen은 "공격해도 리셋"(전투 중 회복=사탕뿐 — 보스 무한탱킹 봉인).

**9. 잔버그 일괄 (~15줄)**: 보스 격파 후 먹구름 장막 잔존(엔딩이 암막 — 1줄) · 스택 99 상한 구매 골드 증발 · mq06 마을 재입장 부슬이 무한 스폰 · 사라진 작물 id 밭 영구 잠금 · beforeunload 저장 · regenTo 데이터 미사용 · 물뿌리개 "3칸"→"십자 5칸" 문구 정정.

**10. 진행 정체 폴백 (~12줄)**: mq06 후 sq02(콩 배달) 미수락이면 활성 퀘스트 0 → 추적 문구가 "다음 부탁: 모모에게 가 보자!"로 폴백 + 요새문 잠금 문구 교정 — 메인 체인 정체 방지.

### POLISH-2 — 상용감·루프 (P1 · ~600줄)
1. **취침 확인+해질녘 (피드백 ④ 응답)**: 침대 탭 즉시 취침 → 확인 선택지("잘 자기 / 아직 안 잘래")로 오조작 차단. 밭일 완료 시 "🌙 오늘 밭일 끝!" 해질녘 신호. 취침 페이드에 **오늘 하루 요약 카드**(처치·골드·수확 — 스타듀식 정산) + "자는 동안 작물 n칸이 자랐다!". *판정: 잠=하루는 교실 통제 장치로 유지(실시간 타이머 기각 — 결석 무벌점·차시 리듬 보존). 대신 '하루'의 보상(리스폰·의뢰 갱신·정예)을 늘려 "자는 게 이득"으로 반전.*
2. **먼별 '해질녘의 우물가'**: appears:"night" 데이터 소생 — "아직 안 잘래" 선택 또는 밭일 완료 시 해질녘 상태(그날 한정), 먼별이 우물가에만 출현(여백⑥ '우물 노랫소리'와 물리적 결합) + 비밀 대사·재료 교환 상점(별조각 가루 5→사탕 3 / 달빛 깃털 3→100G — 재료 골드 싱크).
3. **일일 의뢰판 (클리어 후 루프 본체 · ~180줄)**: 게시판 choice 분기(오늘의 부탁/도감). mq04 후 1건/일 → mq07 후 3건/일. `RPG_DB.dailies` 데이터 주도(학생 확장 슬롯 겸용) — kill/납품 템플릿 9종, day 결정적 시드(재로드 리롤 방지), 보상 합계 60~120G(농사 수입의 절반 미만 가드).
4. **juice 잔여**: 레벨업 연출(라디얼+스탯 표기) · 퀘스트 완료 배너 정식판 · 생활 파티클 3종(수확 팝·물방울·흙) · 기습 연출 차별화·숫자 산포 · 피격 비네트(테두리만·점멸 금지) · 화면 흔들림(보스 한정·4px·토글 필수 — 멀미 이력).
5. **퀘스트 수락 카드**: 보상 사전 노출+클릭 1회(현행 수락 후 대사 3줄 추가 클릭 제거). mq01(자동)은 분기 유지.
6. **온보딩 트리거화**: 미사용 튜토리얼 대사 8줄 소비 — 첫 이동/첫 필드/첫 밭/첫 물주기/첫 수확 이벤트 트리거 배너 + 건물 간판 4개 + 워프 라벨("▶ 이슬별 들판") + mq01 화살표가 촌장집 문을 가리키게(다른 맵 NPC면 포탈 폴백). 기존 세이브는 진행도로 일괄 스킵.
7. **농사 노동 완화**: 리프 2단계 "반짝 물뿌리개"(3×3, 300G) + canUp 일괄 수확(십자) — 40칸 시대 대비. 재료 전부 팔기 버튼.
8. **정예 '별그늘 조각'**: mq07 후 day 시드로 들판/숲길 교대 1마리(tier3 버짓 39.7≤65 검증) — 달빛 깃털 루프. **도감 별점**(10킬 ⭐/30킬 ⭐⭐, 8종 ⭐⭐=칭호 "들판의 파수꾼").
9. **경제 싱크**: 별빛 거름(30G, 성장 +1일, 칸당 1회 플래그 — 이익 증폭 불가 검산 완료) · 사탕 주머니 150G(상한 5→8) · 강화 +3(500G+달빛 깃털 5, mq07 후).
10. **den2 상자 루팅**(1회성 보상 4종) + **학생 승인 몬스터 스폰 슬롯**(현재 승인해도 필드에 안 나옴 — 필드별 stuSlots 3좌표, habitat·tier 일치 로테이션. 수업 목적상 중요).

## B. 렌즈 충돌 조정 기록 (구현 시 이 결정을 따름)
1. 리스폰 시계: 실시간(Date.now) 대신 **playSec** — 교실 공정성(안 켠 시간 무관)·조작 무관. 쿨다운 수치는 스폰 렌즈안(90/150/240).
2. 이름표 위치: **발밑** (위층은 HP바·❗·보스 스프라이트 점유 + NPC 이름=위와 이분법) + UX 렌즈의 역스케일·외곽선 병합.
3. 부활 개선: 체크포인트 부활안 기각 → **별샘 워프망** 채택(상위호환·탐험 편의 겸용).
4. 밤 시스템: night 모드(80줄) 기각 → **해질녘(dusk) 플래그**(마을 한정·그날 소멸) 채택 — 스코프 최소.
5. 사탕: 세 렌즈 합류 — 상한 시행+쿨+HUD+주머니. 기보유 초과분 몰수 금지.
