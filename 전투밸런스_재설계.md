# 전투 밸런스 재설계 — "탄막 피하기 → 신중한 브롤러"

> 작성: 2026-07-05 · 계기: 3v3 플레이 시 "브롤인지 탄막 피하기인지 모호". 근본 전투 느낌 재설계.
> ⚠️ 이 문서의 수치는 전부 **시작값 · 플레이로 튜닝 필수**(밸런스는 굴려야 나옴). 코어 변경이라 영향 큼.
> 기존 상수: playerMaxHp 100 · playerDamage 18 · playerRange 470 · attackCd ~0.42s · moveSpeed 225 · superCharge 100(+20/명중) · gameTime 90.

## 1. 진단 — 왜 탄막 느낌인가
- **기본 공속이 빠름**(~0.42초/발 ≈ 초당 2.4발) × **다발 무기**(별골렘 3발 등) × **6명** → 화면이 총알 폭우.
- 각 탄이 약해서(18) "잘 쏘기"보다 "다 피하기" 게임이 됨 → 탄막 피하기 정체성.

## 2. 방향 — 한 축으로 정리: "느리게 시작 → 처치로 성장"
롤처럼 **초반 공속 느림 → 매치 중 강해짐.** 이 하나로 탄막·성장·섬멸전 버프를 동시에 해결.
- **기본 공속을 늦춘다** → 초반 총알 적음 = 탄막 아님. "한 방 한 방이 의미 있는" 브롤러.
- **처치하면 버프 드랍** → 주우면 공속·공격력 오름 = 롤식 성장. 점점 강해지는 손맛.
- 즉 **탄막 제거(느린 기본) + 성장(버프 드랍)**을 같은 시스템으로.

## 3. 기본 수치 변경 (제안 · 시작값)
| 상수 | 현재 | 제안 | 이유 |
|---|---|---|---|
| 기본 공속(attackCd) | ~0.42s | **~0.72s** (초당 ~1.4발) | 총알 반토막 → 탄막 해소 |
| 기본 공격력 | 18 | **28** | 발수 줄어든 만큼 한 방 강하게(TTK 유지) |
| 다발 무기(spread) | 3발 등 | **발수·산탄 축소** | 화면 정리. 다발은 근접·산탄 정체성만 |
| 이동속도 | 225 | 유지~약간↑ | 느린 공속 = 위치 싸움 → 기동 중요 |
| 궁 게이지(+20/명중) | 5명중 | **명중당 상향(공속 느려짐 보정)** | 궁 회전 유지 |
- TTK 목표: 100HP를 1명이 **~3초**(집중사격), 팀 포커스면 빠르게. "잘 맞히면 죽인다".

## 4. 성장 루프 — 처치 = 버프 드랍 (섬멸전 + 캠페인 공용)
- **적/몬스터 처치 시 그 자리에 버프 오브 드랍.** 주우면 **스택** 획득:
  - 종류(제안): 공속 +8% / 공격력 +10% / 최대체력 +15 (랜덤 or 3종 순환)
  - 최대 **5스택**(폭주 방지).
- **죽어도 내 스택 유지**(리스폰 시 그대로) — 단 **나를 잡은 적이 오브 1개 획득** → 강자 잡으면 보상 = 역전 여지.
- **캠페인에도 적용**: 몬스터 처치→버프. 이게 "느린 기본 공속"을 자동 보정 → 캠페인 따로 크게 안 만져도 됨. (§7 참조)

## 5. 역할 프레임 (공속 느려진 만큼 역할 구분 뚜렷해야)
| 역할 | 체력 | 사거리 | 공격 | 스킬 방향 |
|---|---|---|---|---|
| **탱커** | 높음(×2) | 짧음 | 강함·느림 | 방패·몸빵 | → 별골렘 |
| 근접 딜러 | 중 | 중근접 | 버스트 | 돌진·타격 |
| 원거리 딜러 | 낮음 | 긺 | 정밀 | 견제 |
| 서포터/방해 | 중 | 중 | 약함 | 회복·둔화·CC | → 눈꽃(둔화)·모아(회복) |
- 캐릭 6종 배정 초안: **별골렘=탱커 / 눈꽃=방해 / 모아=서포터 / 나머지(럭키·달이·시고니)=딜러 계열.** (세부는 각 캐릭 킷 보고 확정 — 미결)

## 6. ⭐ 별골렘 재설계 (탱커 확정)
현재: 젤리 3발·단거리·약함 = 정체성 없음("구림"). → **확실한 탱커로.**
| 항목 | 현재 | 제안 |
|---|---|---|
| 최대 체력 | ×1.0 (100) | **×2.0 (≈200)** |
| 이동속도 | — | **느리게**(탱커) |
| 공격 | 3발 약함·중거리 | **짧은 사거리 + 강한 한 방**(근접 압박) |
| X 특수 | (현재 약함) | **의미있는 방패 생성** — 자신/아군 흡수막(에셋 나중) |
| C 궁극 | — | 탱커형(대형 방패·도발·넉백 중 택 — 미결) |
- 방패 X = 탱커의 핵심. 흡수량·지속·쿨은 시작값 잡고 튜닝.

## 7. 캠페인 연동 (필수)
- 기본 공속 느려짐 = 웨이브 클리어 어려워짐. **보정 필요.**
- **1차 보정 = §4 버프 드랍**(몬스터 처치→버프)이 자동으로 완화. 이걸로 상당 부분 해결.
- **부족하면**: 몬스터 HP ~15% ↓ 또는 스폰 수 조정. 스테이지별 재검증.
- 슬라이서/에셋 무관, 순수 수치.

## 8. 미결 (사용자 확정)
- 기본 공속/공격력 최종 목표 TTK(느낌).
- 버프 드랍: 종류(고정 3종 순환 vs 랜덤), 스택 상한, 죽을 때 오브 규칙.
- 캐릭 6종 역할 최종 배정(각 킷 검토 후).
- 별골렘 궁극기 방향.
- 강화 3택1(로그라이트)과 버프 드랍 성장의 관계(둘 다? 하나만?).
