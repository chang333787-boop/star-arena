## 0. 목적
캠페인을 "몬스터 웨이브 20판"에서 **"별빛 세계를 순례하는 원정"**으로. 구역마다 테마(바닥·몬스터 팔레트·보스)가 뚜렷하고, 성장버프로 판마다 강해지는 손맛, 변형 스테이지로 단조로움 제거.

**비목표**: 스테이지 수 변경(20 유지) · 신규 몬스터 제작(13종 재배치만) · 컷신/스토리 텍스트 대량 추가(구역 인트로 1줄만) · 신규 아트.

## 1. 구역 테마 재구성 (핵심)
| 구역 | 테마 | 바닥(floor) | 몬스터 팔레트(주연) | 보스 | 변형 스테이지 |
|---|---|---|---|---|---|
| 1 별빛 초원 | 입문·풀내음 | **grass** | soft_jelly·bounce_bubble·star_bug | **boss_robot** (s5) | s4 🛡 방어전(기존 defend) |
| 2 얼음 동굴 | 미끄럼·수정 | **snow** (+얼음 함정 칸) | ice_bubble·shield_block·cushion_block | **boss_turtle** (s10) | s9 🚪 봉쇄전(기존 blockade) |
| 3 나무 요새 | 좁은 통로·기습 | **wood** | paper_drone·cloud_ghost·heal_fairy | **boss_plane** (s15) | s14 ✨ **거점 사수전(신규)** |
| 4 어둠의 심연 | 최종장·혼합 | **void** | 전 몬스터 혼합 + 정예(elite) | **boss_king** (s20) | s19 ⭐ **별 수호전(신규)** |
- 각 구역 첫 스테이지 진입 시 **구역 인트로 배너**(테마명 + 한 줄, 2.5s): "❄ 얼음 동굴 — 바닥이 미끄러워요!".
- 맵: 기존 mapId 유지하되 **floor 필드를 구역 테마로 교체**(training=grass, crystal_cave=snow, galaxy_garden=wood, 최종=void). 2구역 맵에 얼음 함정 칸 소량 배치(테마 체감).
- 몬스터 팔레트: 스테이지별 스폰 구성을 구역 주연 위주로 재배치(전 종 등장 금지 아님 — 주연 60%+ 권장).

## 2. 신규 변형 스테이지 2종 (신모드 기술 재활용 — 구현 쌈)
### 2-1 ✨ 거점 사수전 (s14) — 점령전 기술 재활용
- 중앙 거점 링(radius 110). **아군이 거점 안에 있어야 게이지 상승**(+8/s), 몬스터가 거점 안에 있으면 정지.
- 게이지 100 = 클리어. 웨이브는 무한 리스폰(잡는 게 목적이 아니라 버티는 것) — 시간압박형.
### 2-2 ⭐ 별 수호전 (s19) — 별모으기 기술 재활용
- 중앙 별광맥에서 별이 8s마다 스폰. 몬스터가 별에 닿으면 **별을 물고 맵 밖으로 도주**(도주 몬스터 = 이속 +20%).
- **별 5개 뺏기면 실패** / 90s 버티면 클리어. 도주 몬스터 우선격파라는 새로운 판단 요구.
- 도주 AI: 별 획득 → 최근접 맵 가장자리로 직행(기존 이동 AI 재사용).

## 3. 성장버프 통합 (E16·GAMEPLAY-1 1D 확정 적용)
- **캠페인 전 스테이지에서 처치→버프 오브 드랍 활성**(섬멸전과 동일 규칙: 공속+8%/공격+10%/체력+15, 최대 5스택).
- **스택은 구역 내 유지, 구역 넘어갈 때 리셋**(구역 = 성장 아크. 밸런스 폭주 방지 + "구역마다 다시 빌드" 리듬). ※ 오픈 퀘스천 §6-1.
- 강화 3택1(기존 로그라이트)은 그대로 — 버프 오브(판 내 성장)와 층이 다름(런 전체 성장).

## 4. 난이도 곡선 (재설계 전투 기준 재조정)
**원칙: 입문은 쉽게, 상승은 완만하게** — 모든 게임의 정석대로 1구역은 "이길 수 있어서 재밌는" 구간. 급턱(구역 경계에서 확 어려워짐) 금지 — **스테이지 단위 선형 보간**으로 부드럽게.
| 스테이지 | HP 배율 | 스폰 수 배율 | 비고 |
|---|---|---|---|
| s1~5 (초원) | **0.75 → 0.95** | 0.85 → 1.0 | 입문 — 성취감 우선, 조작·성장버프 학습 |
| s6~10 (동굴) | 0.95 → 1.1 | 1.0 → 1.1 | 여기서부터 슬슬 긴장 |
| s11~15 (요새) | 1.1 → 1.3 | 1.1 → 1.2 | |
| s16~20 (심연) | 1.3 → 1.5 | 1.2 → 1.3 | 최종장 — 도전 |
- 배율은 구역 경계에서도 **연속**(s5 끝값 = s6 시작값) — 완만한 단일 곡선. 시작값 — B콘솔 그룹 "캠페인" 등록, 난이도(쉬움/보통/어려움) 배율은 그 위에 곱(기존 유지).
- 몬스터 구성도 곡선과 동행: 1구역은 근접 위주(soft_jelly 등), **원거리(paper_drone)는 2구역부터** 본격 — 위협 종류도 서서히 늘림.
- **클리어 가능성 하니스**: 각 구역 대표 스테이지(s4·s9·s14·s19·s20) 자동 시뮬 통과 필수.

## 5. 보상·연출
- 구역 클리어(보스 격파) 보너스: +60G·exp+30 (기존 판별 보상에 추가). 20 클리어 = 원정 완료 배너 + 누적 통계(총 처치·시간).
- 체크포인트(12.6) 유지 — 구역 시작 스테이지가 체크포인트.
- 보스전 연출 소폭: 보스 HP 50% 시 **분노 페이즈**(이속·발사 +15%, 색 플래시) — 패턴 추가 없이 긴장감(시작값, 콘솔 등록).

## 6. 오픈 퀘스천 (기본값으로 진행, 결과 문서에 표기)
1. 버프 스택 리셋 단위 — 기본: **구역별 리셋** (대안: 20판 내내 유지 / 매판 리셋).
2. 별 수호전 실패 조건 5개 — 기본 5 (콘솔 조정 가능).
3. 구역 인트로 배너 문구 — 기본: 위 표 스타일 1줄(교사가 수정할 필요 없는 수준).

## 7. 수용 기준 (AC)
- [ ] 4구역 테마(바닥·팔레트·보스) 적용 — 구역별 스크린샷.
- [ ] 신규 변형 2종(거점 사수·별 수호) 클리어/실패 양쪽 동작 + 하니스.
- [ ] 성장버프 캠페인 활성 + 구역 리셋 동작.
- [ ] 클리어 가능성 시뮬(s4·s9·s14·s19·s20) PASS + 기존 하니스 전종 회귀.
- [ ] 구역 인트로·보스 분노·원정 완료 연출 확인. 콘솔 에러 0 · 바이트 동일.

## 8. 금지·중단

- 중단조건: 클리어 시뮬이 튜닝으로 해결 안 되는 구조적 실패 / 거점·별 기술 재활용이 PVE 루프와 충돌 → 기록 후 보고.
